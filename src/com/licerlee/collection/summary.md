
# 总结

- 容器可以自动调整大小，数组不能调整大小（不能add，delete。否则Unsupported Operation Exceptin）
- 容器主要分为Collection和Map
- Collection 常用 List有序的，Set无序不重复的，queue排队确定顺序
- 容器打印Arrays.toString()
- 迭代器Iterator只能向前移动，ListIterator可以双向移动
